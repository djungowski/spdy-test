Trojaner auf Wetter.com: So schützen Sie sich


Fotostrecke: Echte Wetter.com-Alternativen für PC, Handy und Tablet

Malware-Gefahr: Nach einem Angriff verteilte Wetter.com Malware an Besucher.
Schwarze Wolken bei Wetter.com: Die Webseite wurde Opfer einer Hackerattacke und verteilte in den letzten zwei Tagen Trojaner und andere Schadsoftware an Besucher. Der Angriff nutzt eine bekannte Schwachstelle im Anzeigeserver OpenX. So konnten Angreifer eigenen JavaScript-Code einfügen, der wiederum weiteren Code nachlädt und so Besucher mit unterschiedlichen Schädlingen infiziert.

Verwirrung um Stand der Dinge


Verwirrung kam auf, als es bereits am Montag von Wetter.com hieß, dass das Problem behoben worden sei. Auf der Webseite gab es nur kurz einen Hinweis auf einen Fehler, konkrete Angaben zum Angriff Fehlanzeige. Doch auch am gestrigen Dienstag verteilte die Webseite fleißig Malware an die Besucher, unter anderen Scareware und den BKA-Trojaner.

So schützen Sie sich


Browser, die Googles Safe Browsing API nutzen, etwa Firefox und Chrome, blenden beim Besuch von betroffenen Seiten zumindest eine Warnung ein. Das ist aber nur ein Hinweis und kein ausreichender Schutz. Hilfreich sind in diesem Fall Scriptblocker wie NoScript oder Click-to-Play im Browser und natürlich aktuelle Software, denn ein Großteil der Malware setzt auf Lücken in Plug-ins.

Virenschutz auch für den Notfall


Sicherer können sich Besucher der Seite mit einem aktuellen Virenschutz fühlen, denn der sollte im Hintergrund nachgeladene Schädlinge erkennen. Da aber nicht klar ist, welche Malware genau verteilt wurde, sollten Nutzer, die Wetter.com in den letzten Tagen besucht haben, auf jeden Fall einen Offline-Scan des betroffenen Rechners machen. Erstellen Sie sich dazu ein Notfall-Medium auf einem sauberen Rechner, beispielsweise mit SARDU. Wie das geht, erklärt die Fotostrecke. Eine gute Alternative zu Wetter.com ist zum Beispiel Wetter.de.  (jg)
