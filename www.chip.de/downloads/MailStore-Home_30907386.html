MailStore Home


 

Mit dem kostenlosen "MailStore Home" archivieren und sichern Sie all Ihre E-Mails in nur einem Programm.
 
Nutzt man mehrere E-Mail-Clients, verliert man schnell den Überblick über seine Nachrichten. Damit Sie ständig alles auf einen Blick sehen und keine Nachricht verloren geht, können Sie Ihre Mail mit Hilfe der Freeware "MailStore Home" archivieren.

Nach dem Backup können Sie besonders wichtige E-Mails direkt aus dem Tool heraus auf CD/DVD brennen. Des weiteren werden Anhänge komprimiert, um Speicherplatz zu sparen. "MailStore Home" hat auch eine sehr schnelle Suche integriert, die das Auffinden von E-Mails erleichtert.

Eine weitere komfortable Funktion von "MailStore Home" ist die Möglichkeit, E-Mails direkt aus dem Tool heraus zu beantworten. Hierfür öffnet das Tool dann Ihren standardmäßigen E-Mail-Client. Unterstützt werden alle gängigen Programme und Postfächer, etwa Microsoft Outlook, Outlook Express, Windows Live Mail, Mozilla Thunderbird, Webmailer wie Google Mail, Exchange Server, sowohl POP und IMAP.

In der neuen Version bietet Mailstore Home eine überarbeitetes Benutzerinterface und eine bessere Unterstützung für die Microsoft Exchange Web Services (EWS). Alle Änderungen und Neuerungen dieser Version 7.0.x finden Sie im offiziellen Changelog auf der Homepage des Herstellers.

Fazit: Sehr praktische Anwendung, die viel Zeit spart und zudem kostenlos erhältlich ist.

Hinweis: Auf der Hersteller-Webseite gibt es weiterhin eine 300-Euro teure Server-Version, die sich speziell an Unternehmen richtet.
